# UKFC-bot

## 部署
### 启动拉格朗日服务器
[快速部署 & 配置](https://lagrangedev.github.io/Lagrange.Doc/v1/Lagrange.OneBot/Config/)
### 部署本项目
```
# npm
npm i

# yarn
yarn
```
项目内自行创建 .env 存放账号信息
编译启动(先启动 Lagrange)

```
npx tsc
node ./dist/main.js
```
