# UKFC-bot

## 部署
### 启动拉格朗日服务器
[快速部署 & 配置](https://lagrangedev.github.io/Lagrange.Doc/v1/Lagrange.OneBot/Config/)
### 部署本项目
```
# npm
npm install lagrange.onebot

# yarn
yarn add lagrange.onebot
```
```
npx tsc
node ./dist/main.js
```

或者直接docker搭好分别起两个服务即可，需要更改qq号
